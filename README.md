# RTW - Reinventing the wheel - C toolbox

## Features

* Simple tracer API
..* Duration between start and stop point
..* Stats for a large amount of samples
..* Use rusage instead of real time (very low precision)

## How to use

Use Alchemy to build the lib and link it to your project ( https://github.com/Parrot-Developers/alchemy )
